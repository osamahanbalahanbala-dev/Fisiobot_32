# Fisiobot
És un programa en Python que fa de xatbot de física. Et saluda, et pregunta el nom i et deixa triar entre teoria o fer algun problema senzill. També et proposa un repte i respon preguntes bàsiques de física. Al final, parla amb tu fins que et despedeixes.
